Protocole Factuel v1.1

Comment on travaille

Factuel est un journal numérique entièrement rédigé par intelligence artificielle selon un protocole éditorial strict et public. Voici comment chaque article est produit.

"Juste les faits. Aucun parti pris." — Ce n'est pas un slogan. C'est une contrainte technique inscrite dans le code.

1. Sources RSS institutionnelles

Le pipeline collecte chaque jour les flux RSS de sources officielles et scientifiques : Le Monde (Science, Économie, Santé), CNRS Actualités, Futura Sciences, Vie Publique, INSEE, Eurostat. Aucun media d'opinion, aucun réseau social.

2. Filtre éditorial automatique (Score v2)

Chaque article candidat est scoré de 0 à 100 :

  • +35 — Source majeure (INSEE, CNRS, INSERM, revue peer-reviewed)
  • +15 — Média de référence (AFP, Reuters, Le Monde)
  • +15 — Mots-clés de confiance (données, rapport, étude, statistique)
  • +15 — Fraîcheur absolue (publié il y a moins de 12h)
  • +20 — Densité (contenu riche ET source qualifiée)
  • −50 — Sujet redondant (même thématique dans les 48h)
  • REJET — Blacklist (guerre, terrorisme, people, publicité, sondage d'opinion)

Un quota de 3 articles maximum par catégorie est appliqué par cycle pour garantir la diversité éditoriale.

3. Génération par IA (Llama 3.3 70B)

Les articles retenus sont envoyés à un modèle de langage (Groq / Llama-3.3-70b) avec un prompt système strict : chaque fait doit être attribué à une institution avec "Selon [institution]", minimum 4 sources identifiées, 700 mots minimum, aucune opinion, aucune conclusion politique. Le modèle ne peut pas spéculer ni interpréter.

4. Validation automatique

Avant publication, chaque article est validé par le pipeline : minimum 3 sources identifiées, corps de texte supérieur à 600 caractères. Les articles qui ne passent pas la validation sont automatiquement rejetés et non publiés.

5. Publication automatique

Le pipeline tourne deux fois par jour : 07h00 (10 articles) et 18h30 (5 articles). Aucune intervention humaine dans la sélection ou la rédaction. Le code source complet est disponible sur GitHub.
